# Hourglass Animation

A minimal, black and white hourglass animation built with HTML5 Canvas.

## Live Demo
[https://exzenter.github.io/hourglass-svg/](https://exzenter.github.io/hourglass-svg/)

## Features
- **Accurate Physics**: Sand falls with particle simulation and accumulation.
- **Visuals**: Clean, thick-stroke transparent design matching the reference style.
- **Looping**: Automatically rotates and morphs the sand pile to reset the animation seamlessly.
- **Interactive**: Pure JavaScript and Canvas implementation, no external dependencies.
